Example Device Name
===================
ESPNA1VDBP01

Device Naming Convention
========================
Country Code
- US, ES, FR, UK, etc


Location Code (Nearest Airport)
- LAX, PNA, CDG, LHR, etc


Unique Site Code (Datacenter, Street, Building ID)
- 1, A, etc

Device Type
- V=Virtual
- C=Cluster
- P=Physical
- O=Outsourced or vendor supported system


Device Role
- DC=Domain Controller
- FS=File Server
- PS=Print Server
- AS=Application Server
- WS=Web Server
- ORA=Oracle Database
- SQL=SQL Database
- DB=Database
- EXH=Microsoft Exchange
- CTX=Citrix Server
- ESX=VMware ESX Server
- SW=Switch
- RT=Router
- FW=Firewall
- etc


Service level
- P=Production
- D=Development
- T=Test
- S=Stagging
- etc


Sequential ID
- 01, 001, etc
